# Student Result Prediction

This project predicts whether a student will pass or fail based on:
- Hours Studied
- Attendance

## Model Used
- Logistic Regression

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Google Colab

## Features
- Predicts student performance (Pass/Fail)
- Uses machine learning model
- Simple and easy to use

## Output
- 1 = Pass
- 0 = Fail
- ## Results
- Model Accuracy: 0.8
